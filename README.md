## 🚀 Getting Started

### Prerequisites

- Java JDK 17 or later
- Any Java IDE (IntelliJ IDEA, Eclipse, VS Code)
- Git (optional)

### Clone the Repository

```bash
git clone https://github.com/your-username/Basics-to-Advance-Java.git
```

### Navigate to the Project

```bash
cd Basics-to-Advance-Java
```

### Compile a Java Program

```bash
javac FileName.java
```

### Run the Program

```bash
java FileName
```
